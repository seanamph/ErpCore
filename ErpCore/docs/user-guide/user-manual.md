# 使用者手冊

## 概述

本文檔提供 ErpCore 系統的使用者操作指南。

## 系統登入

### 登入步驟
1. 開啟瀏覽器，輸入系統網址
2. 輸入使用者帳號和密碼
3. 點擊「登入」按鈕

### 忘記密碼
1. 點擊「忘記密碼」連結
2. 輸入註冊的電子郵件
3. 按照郵件指示重置密碼

## 主要功能

### 系統管理
- **使用者管理**: 管理系統使用者
- **角色管理**: 管理使用者角色
- **權限管理**: 設定系統權限

### 基本資料管理
- **參數設定**: 設定系統參數
- **地區設定**: 管理地區資料
- **組織架構**: 管理部門和組織

### 進銷存管理
- **商品管理**: 管理商品資料
- **庫存管理**: 查詢和管理庫存
- **進銷存報表**: 查看進銷存報表

### 採購管理
- **採購單管理**: 建立和管理採購單
- **驗收作業**: 處理採購驗收
- **採購報表**: 查看採購報表

### 調撥管理
- **調撥單管理**: 建立和管理調撥單
- **驗收作業**: 處理調撥驗收
- **驗退作業**: 處理調撥驗退

## 操作指南

### 新增資料
1. 點擊「新增」按鈕
2. 填寫表單資料
3. 點擊「儲存」按鈕

### 查詢資料
1. 輸入查詢條件
2. 點擊「查詢」按鈕
3. 查看查詢結果

### 修改資料
1. 在列表中點擊「編輯」
2. 修改資料
3. 點擊「儲存」按鈕

### 刪除資料
1. 在列表中點擊「刪除」
2. 確認刪除操作

## 常見問題

### 無法登入
- 檢查帳號和密碼是否正確
- 確認帳號是否被停用
- 聯絡系統管理員

### 權限不足
- 確認使用者角色權限
- 聯絡系統管理員調整權限

### 資料無法儲存
- 檢查必填欄位是否填寫
- 檢查資料格式是否正確
- 查看錯誤訊息提示

## 技術支援

如遇到問題，請聯絡系統管理員或技術支援團隊。
