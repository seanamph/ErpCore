# SYSN810-SYSN830 - 財務其他報表系列 開發計劃

## 一、功能概述

### 1.1 功能說明
- **功能代碼**: SYSN810-SYSN830系列
- **功能名稱**: 財務其他報表系列
- **功能描述**: 提供財務其他報表的查詢、列印、匯出功能，包含其他報表類型、其他報表參數等
- **參考舊程式**: 
  - `WEB/IMS_CORE/ASP/SYSN000/SYSN810_*.ASP`
  - `WEB/IMS_CORE/ASP/SYSN000/SYSN830_*.ASP`

### 1.2 業務需求
- 支援其他報表類型
- 支援其他報表參數設定
- 支援其他報表列印與匯出

## 三、後端 API 設計

### 3.1 API 端點列表

#### 3.1.1 查詢財務其他報表
- **HTTP 方法**: `GET`
- **路徑**: `/api/v1/financial-reports/others`
- **說明**: 查詢財務其他報表資料

#### 3.1.2 匯出財務其他報表
- **HTTP 方法**: `POST`
- **路徑**: `/api/v1/financial-reports/others/export`
- **說明**: 匯出財務其他報表（Excel、PDF）
