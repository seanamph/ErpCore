# SYSN250 - 財務交易報表系列 開發計劃

## 一、功能概述

### 1.1 功能說明
- **功能代碼**: SYSN250系列
- **功能名稱**: 財務交易報表系列
- **功能描述**: 提供財務交易報表的查詢、列印、匯出功能，包含交易明細表、交易統計表、交易彙總表等多種報表類型
- **參考舊程式**: 
  - `WEB/IMS_CORE/ASP/SYSN000/SYSN250_PR.ASP` (報表)

### 1.2 業務需求
- 支援多種財務交易報表類型
- 支援報表列印與匯出
- 支援報表參數設定

## 三、後端 API 設計

### 3.1 API 端點列表

#### 3.1.1 查詢財務交易報表
- **HTTP 方法**: `GET`
- **路徑**: `/api/v1/financial-transactions/reports`
- **說明**: 查詢財務交易報表資料

#### 3.1.2 匯出財務交易報表
- **HTTP 方法**: `POST`
- **路徑**: `/api/v1/financial-transactions/reports/export`
- **說明**: 匯出財務交易報表（Excel、PDF）
