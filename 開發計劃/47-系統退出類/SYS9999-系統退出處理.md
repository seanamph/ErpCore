# SYS9999 - 系統退出處理 開發計劃

## 一、功能概述

### 1.1 功能說明
- **功能代碼**: SYS9999
- **功能名稱**: 系統退出處理
- **功能描述**: 提供系統退出處理功能，包含Session清理、資源釋放、日誌記錄等系統退出相關處理
- **參考舊程式**: 
  - `WEB/IMS_CORE/ASP/SYS9999/SYS9999.ASP`
  - `IMS3/HANSHIN/IMS3/ASP/SYS9999/SYS9999.ASP`

### 1.2 業務需求
- 清理使用者Session
- 釋放系統資源
- 記錄系統退出日誌
- 處理未完成的交易
- 清理臨時檔案
- 通知相關系統

### 2.1 相關資料表

#### 2.1.1 `SystemExitLogs` - 系統退出日誌
```sql
CREATE TABLE [dbo].[SystemExitLogs] (
    [LogId] BIGINT NOT NULL PRIMARY KEY IDENTITY(1,1),
    [UserId] NVARCHAR(50) NULL,
    [ExitType] NVARCHAR(20) NOT NULL, -- NORMAL, TIMEOUT, FORCE, ERROR
    [ExitTime] DATETIME2 NOT NULL DEFAULT GETDATE(),
    [IpAddress] NVARCHAR(50) NULL,
    [UserAgent] NVARCHAR(500) NULL,
    [Reason] NVARCHAR(500) NULL,
    [Details] NVARCHAR(MAX) NULL,
    [CreatedAt] DATETIME2 NOT NULL DEFAULT GETDATE(),
    CONSTRAINT [FK_SystemExitLogs_Users] FOREIGN KEY ([UserId]) REFERENCES [dbo].[Users] ([UserId])
);

-- 索引
CREATE NONCLUSTERED INDEX [IX_SystemExitLogs_UserId] ON [dbo].[SystemExitLogs] ([UserId]);
CREATE NONCLUSTERED INDEX [IX_SystemExitLogs_ExitTime] ON [dbo].[SystemExitLogs] ([ExitTime]);
CREATE NONCLUSTERED INDEX [IX_SystemExitLogs_ExitType] ON [dbo].[SystemExitLogs] ([ExitType]);
```

| CreatedAt | DATETIME2 | - | NO | GETDATE() | 建立時間 | - |

## 三、後端 API 設計

### 3.1 API 端點列表

#### 3.1.1 系統退出處理
- **HTTP 方法**: `POST`
- **路徑**: `/api/v1/system/exit`
- **說明**: 處理系統退出，清理資源並記錄日誌
- **請求格式**:
  ```json
  {
    "exitType": "NORMAL",
    "reason": "使用者主動退出",
    "details": {}
  }
  ```
- **回應格式**:
  ```json
  {
    "success": true,
    "code": 200,
    "message": "系統退出處理完成",
    "data": null,
    "timestamp": "2024-01-01T00:00:00Z"
  }
  ```

#### 3.1.2 查詢系統退出日誌
- **HTTP 方法**: `GET`
- **路徑**: `/api/v1/system/exit-logs`
- **說明**: 查詢系統退出日誌列表
- **請求參數**:
  ```json
  {
    "pageIndex": 1,
    "pageSize": 20,
    "userId": "",
    "exitType": "",
    "startDate": "",
    "endDate": ""
  }
  ```
- **回應格式**:

### 3.2 後端實作類別

#### 3.2.1 Controller: `SystemController.cs`
```csharp
namespace RSL.IMS3.Api.Controllers
{
    [ApiController]
    [Route("api/v1/system")]
    [Authorize]
    public class SystemController : ControllerBase
    {
        private readonly ISystemService _systemService;
        
        public SystemController(ISystemService systemService)
        {
            _systemService = systemService;
        }
        
        [HttpPost("exit")]
        public async Task<ActionResult<ApiResponse>> Exit([FromBody] SystemExitDto dto)
        {
            var userId = User.FindFirst(ClaimTypes.NameIdentifier)?.Value;
            var ipAddress = HttpContext.Connection.RemoteIpAddress?.ToString();
            await _systemService.ProcessExitAsync(userId, dto, ipAddress);
            return Ok(new ApiResponse { Success = true, Message = "系統退出處理完成" });
        }
        
        [HttpGet("exit-logs")]
        public async Task<ActionResult<ApiResponse<PagedResult<SystemExitLogDto>>>> GetExitLogs([FromQuery] SystemExitLogQueryDto query)
        {
            var result = await _systemService.GetExitLogsAsync(query);
            return Ok(new ApiResponse<PagedResult<SystemExitLogDto>> { Success = true, Data = result });
        }
    }
}
```

#### 3.2.2 Service: `SystemService.cs`
```csharp

    }
```

### 4.3 API 呼叫 (`system.api.ts`)
```typescript
import request from '@/utils/request';

export interface SystemExitDto {
  exitType: 'NORMAL' | 'TIMEOUT' | 'FORCE' | 'ERROR';
  reason?: string;
  details?: Record<string, any>;
}

export interface SystemExitLogQueryDto {
  pageIndex: number;
  pageSize: number;
  userId?: string;
  exitType?: string;
  startDate?: string;
  endDate?: string;
}

export interface SystemExitLogDto {
  logId: number;
  userId: string;
  exitType: string;
  exitTime: string;
  ipAddress: string;
  reason?: string;
}

// 系統退出處理
export const processSystemExit = (dto: SystemExitDto) => {
  return request.post<ApiResponse>('/api/v1/system/exit', dto);
};

// 查詢系統退出日誌
export const getExitLogs = (query: SystemExitLogQueryDto) => {
  return request.get<ApiResponse<PagedResult<SystemExitLogDto>>>('/api/v1/system/exit-logs', { params: query });
};
```
