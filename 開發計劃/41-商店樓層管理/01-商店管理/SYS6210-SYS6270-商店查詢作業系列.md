# SYS6210-SYS6270 - 商店查詢作業系列 開發計劃

## 一、功能概述

### 1.1 功能說明
- **功能代碼**: SYS6210-SYS6270 系列
- **功能名稱**: 商店查詢作業系列
- **功能描述**: 提供商店資料的查詢功能，包含多種查詢條件、排序、分頁、匯出等功能，支援商店基本資料、商店狀態、商店類型、樓層、區域等條件查詢
- **參考舊程式**: 
  - `WEB/IMS_CORE/ASP/SYS6000/SYS6210_FQ.ASP` (查詢)
  - `WEB/IMS_CORE/ASP/SYS6000/SYS6215_FQ.ASP` (查詢)
  - `WEB/IMS_CORE/ASP/SYS6000/SYS6220_FQ.ASP` (查詢)
  - `WEB/IMS_CORE/ASP/SYS6000/SYS6225_FQ.ASP` (查詢)
  - `WEB/IMS_CORE/ASP/SYS6000/SYS6230_FQ.ASP` (查詢)
  - `WEB/IMS_CORE/ASP/SYS6000/SYS6240_FQ.ASP` (查詢)
  - `WEB/IMS_CORE/ASP/SYS6000/SYS6250_FQ.ASP` (查詢)
  - `WEB/IMS_CORE/ASP/SYS6000/SYS6260_FQ.ASP` (查詢)
  - `WEB/IMS_CORE/ASP/SYS6000/SYS6270_FQ.ASP` (查詢)

### 1.2 業務需求
- 提供商店資料的查詢功能
- 支援多種查詢條件組合
- 支援查詢結果排序
- 支援查詢結果分頁顯示
- 支援查詢結果匯出（Excel、PDF）
- 支援商店狀態篩選
- 支援商店類型篩選
- 支援樓層篩選
- 支援區域篩選
- 支援日期範圍查詢
- 支援關鍵字搜尋

## 三、後端 API 設計

### 3.1 API 端點列表

#### 3.1.1 查詢商店列表
- **HTTP 方法**: `GET`
- **路徑**: `/api/v1/shop-floors/query`
- **說明**: 查詢商店列表，支援多種查詢條件、排序、分頁
- **請求參數**:
  ```json
  {
    "pageIndex": 1,
    "pageSize": 20,
    "sortField": "ShopId",
    "sortOrder": "ASC",
    "filters": {
      "shopId": "",
      "shopName": "",
      "shopType": "",
      "status": "",
      "city": "",
      "zone": "",
      "floorId": "",
      "posEnabled": null,
      "openDateFrom": "",
      "openDateTo": "",
      "keyword": ""
    }
  }
  ```
- **回應格式**:
  ```json
  {
    "success": true,
    "code": 200,
    "message": "查詢成功",
    "data": {
      "items": [
        {
          "tKey": 1,
          "shopId": "SHOP001",
          "shopName": "測試商店",
          "shopNameEn": "Test Shop",
          "floorId": "FLOOR001",
          "floorName": "一樓",
          "shopType": "TYPE001",
          "address": "測試地址",
          "city": "台北市",
          "zone": "信義區",
          "phone": "02-12345678",
          "status": "A",
          "openDate": "2024-01-01",
          "posEnabled": true
        }
      ],
      "totalCount": 100,
      "pageIndex": 1,
      "pageSize": 20,
      "totalPages": 5
    },
    "timestamp": "2024-01-01T00:00:00Z"
  }
  ```

#### 3.1.2 匯出商店查詢結果
- **HTTP 方法**: `POST`
- **路徑**: `/api/v1/shop-floors/export`
- **說明**: 匯出商店查詢結果為Excel或PDF
- **請求格式**:
  ```json
  {
    "format": "Excel", // Excel 或 PDF
    "filters": {
      "shopId": "",
      "shopName": "",
      "shopType": "",
      "status": "",
      "city": "",
      "zone": "",
      "floorId": ""
    }
  }
  ```
- **回應格式**: 返回檔案下載連結或直接下載

### 4.3 API 呼叫 (`shopFloorQuery.api.ts`)
```typescript
import request from '@/utils/request';

export interface ShopFloorQueryDto {
  pageIndex: number;
  pageSize: number;
  sortField?: string;
  sortOrder?: 'ASC' | 'DESC';
  filters?: {
    shopId?: string;
    shopName?: string;
    shopType?: string;
    status?: string;
    city?: string;
    zone?: string;
    floorId?: string;
    posEnabled?: boolean;
    openDateFrom?: string;
    openDateTo?: string;
    keyword?: string;
  };
}

export interface ShopFloorExportDto {
  format: 'Excel' | 'PDF';
  filters?: {
    shopId?: string;
    shopName?: string;
    shopType?: string;
    status?: string;
    city?: string;
    zone?: string;
    floorId?: string;
  };
}

// API 函數
export const queryShopFloors = (query: ShopFloorQueryDto) => {
  return request.post<ApiResponse<PagedResult<ShopFloorDto>>>('/api/v1/shop-floors/query', query);
};

export const exportShopFloors = (dto: ShopFloorExportDto) => {
  return request.post('/api/v1/shop-floors/export', dto, {
    responseType: 'blob'
  });
};
```

## 六、注意事項

### 6.1 效能
- 大量資料查詢必須使用分頁
- 必須建立適當的索引
- 複雜查詢必須優化SQL語句
- 匯出大量資料時必須使用非同步處理

### 6.2 資料驗證
- 查詢條件必須驗證
- 日期範圍必須驗證
- 關鍵字搜尋必須處理特殊字元

### 6.3 業務邏輯
- 查詢結果必須根據使用者權限過濾
- 匯出功能必須記錄操作日誌
- 查詢條件必須支援模糊搜尋
