# SYS0340 - 使用者欄位權限設定 開發計劃

## 一、功能概述

### 1.1 功能說明
- **功能代碼**: SYS0340
- **功能名稱**: 使用者欄位權限設定
- **功能描述**: 提供使用者欄位權限設定的查詢、設定功能，用於控制使用者對資料庫欄位的存取權限
- **參考舊程式**: 
  - `WEB/IMS_CORE/ASP/SYS0000/SYS0340_FQ.ASP` (查詢)
  - `WEB/IMS_CORE/ASP/SYS0000/SYS0340_USER_LIST.asp` (使用者列表)
  - `WEB/IMS_CORE/ASP/SYS0000/SYS0340_SET_FIELD_AC.asp` (設定欄位權限)

### 1.2 業務需求
- 管理使用者對資料庫欄位的存取權限
- 支援依資料庫、表格、欄位設定權限
- 支援使用者欄位權限的查詢
- 支援欄位權限的設定與修改
- 支援多資料庫環境
- **重要**: 當設定使用者直接權限時，需移除該使用者的角色權限對應（UserRoles表中的記錄）

## 二、資料庫設計 (Schema)

### 2.1 主要資料表: `UserFieldPermissions`

```sql
CREATE TABLE [dbo].[UserFieldPermissions] (
    [Id] UNIQUEIDENTIFIER NOT NULL PRIMARY KEY DEFAULT NEWID(),
    [UserId] NVARCHAR(50) NOT NULL,
    [DbName] NVARCHAR(100) NOT NULL,
    [TableName] NVARCHAR(100) NOT NULL,
    [FieldName] NVARCHAR(100) NOT NULL,
    [PermissionType] NVARCHAR(20) NOT NULL, -- READ, WRITE, HIDE
    [CreatedBy] NVARCHAR(50) NULL,
    [CreatedAt] DATETIME2 NOT NULL DEFAULT GETDATE(),
    [UpdatedBy] NVARCHAR(50) NULL,
    [UpdatedAt] DATETIME2 NOT NULL DEFAULT GETDATE(),
    CONSTRAINT [FK_UserFieldPermissions_Users] FOREIGN KEY ([UserId]) REFERENCES [dbo].[Users] ([UserId]) ON DELETE CASCADE,
    CONSTRAINT [UK_UserFieldPermissions] UNIQUE ([UserId], [DbName], [TableName], [FieldName])
);

-- 索引
CREATE NONCLUSTERED INDEX [IX_UserFieldPermissions_UserId] ON [dbo].[UserFieldPermissions] ([UserId]);
CREATE NONCLUSTERED INDEX [IX_UserFieldPermissions_DbTable] ON [dbo].[UserFieldPermissions] ([DbName], [TableName]);
```

### 2.3 資料字典

| 欄位名稱 | 資料類型 | 長度 | 允許NULL | 預設值 | 說明 | 備註 |

| CreatedBy | NVARCHAR | 50 | YES | - | 建立者 | - |
| CreatedAt | DATETIME2 | - | NO | GETDATE() | 建立時間 | - |
| UpdatedBy | NVARCHAR | 50 | YES | - | 更新者 | - |
| UpdatedAt | DATETIME2 | - | NO | GETDATE() | 更新時間 | - |

## 六、注意事項

### 6.1 安全性
- 必須實作權限檢查
- 敏感欄位必須有適當的權限控制
- 必須驗證使用者權限

### 6.2 效能
- 大量資料查詢必須使用分頁
- 必須建立適當的索引
- 必須使用快取機制

### 6.3 資料驗證
- 使用者代碼必須存在
- 資料庫、表格、欄位必須存在
- 權限類型必須在允許範圍內

### 6.4 業務邏輯
- 同一使用者對同一欄位只能有一個權限設定
- 刪除權限前必須檢查是否有相關資料
- 權限變更必須記錄
- **重要**: 設定使用者直接權限時，需移除該使用者的角色權限對應（UserRoles表中的記錄）
