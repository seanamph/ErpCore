# SYS0750 - 角色之使用者列表 開發計劃

## 一、功能概述

### 1.1 功能說明
- **功能代碼**: SYS0750
- **功能名稱**: 角色之使用者列表
- **功能描述**: 查詢指定角色下，哪些使用者擁有該角色，以報表形式顯示使用者基本資訊
- **參考舊程式**: 
  - `WEB/IMS_CORE/ASP/SYS0000/SYS0750_PR.asp` (報表)
  - `WEB/IMS_CORE/ASP/SYS0000/SYS0750_FQ.asp` (查詢)
  - `WEB/IMS_CORE/ASP/SYS0000/SYS0750_FD.asp` (刪除)

### 1.2 業務需求
- 支援依角色代碼查詢使用者列表
- 顯示角色基本資訊（角色代碼、角色名稱）
- 顯示擁有該角色的使用者列表（使用者代碼、使用者名稱、使用者型態、帳號狀態、職稱、組織）
- 支援報表列印功能
- 支援報表匯出功能（Excel、PDF）
- 支援刪除使用者角色對應（可選功能）

```sql
CREATE TABLE [dbo].[UserRoles] (
    [UserId] NVARCHAR(50) NOT NULL,

-- 索引
CREATE NONCLUSTERED INDEX [IX_UserRoles_RoleId] ON [dbo].[UserRoles] ([RoleId]);
```

### 2.2 查詢視圖 (View)

為了方便查詢，建議建立以下視圖：

#### 2.2.1 `V_RoleUsers` - 角色使用者視圖
```sql
CREATE VIEW [dbo].[V_RoleUsers] AS
SELECT 
    R.RoleId AS ROLE_ID,
    R.RoleName AS ROLE_NAME,
    U.UserId AS USER_ID,
    U.UserName AS USER_NAME,
    U.UserType AS USER_TYPE,
    U.Status AS STATUS,
    U.Title AS TITLE,
    U.OrgId AS ORG_ID,
    UR.CreatedBy AS CREATED_BY,
    UR.CreatedAt AS CREATED_AT
FROM [dbo].[UserRoles] UR
INNER JOIN [dbo].[Roles] R ON UR.RoleId = R.RoleId
INNER JOIN [dbo].[Users] U ON UR.UserId = U.UserId;
```

## 三、後端 API 設計

### 3.1 API 端點列表

#### 3.1.1 查詢角色之使用者列表
- **HTTP 方法**: `GET`
- **路徑**: `/api/v1/role-users/list`
- **說明**: 查詢指定角色下，哪些使用者擁有該角色
- **請求參數**:
  ```json
  {
    "roleId": "string"        // 角色代碼（必填）
  }
  ```
- **回應格式**:
  ```json
  {
    "success": true,
    "code": 200,
    "message": "查詢成功",
    "data": {
      "roleId": "ROLE001",
      "roleName": "系統管理員",
      "users": [
        {
          "userId": "U001",
          "userName": "張三",
          "userType": "1",
          "userTypeName": "公司人員",
          "status": "A",
          "statusName": "啟用",
          "title": "經理",
          "orgId": "ORG001",
          "orgName": "資訊部"
        }
      ]
    },
    "timestamp": "2024-01-01T00:00:00Z"
  }
  ```

#### 3.1.2 刪除使用者角色對應
- **HTTP 方法**: `DELETE`
- **路徑**: `/api/v1/role-users/{roleId}/{userId}`
- **說明**: 刪除指定使用者與角色的對應關係
- **路徑參數**:
  - `roleId`: 角色代碼
  - `userId`: 使用者代碼
- **回應格式**:
  ```json
  {
    "success": true,
    "code": 200,
    "message": "刪除成功",
    "data": null,
    "timestamp": "2024-01-01T00:00:00Z"
  }
  ```

#### 3.1.3 批次刪除使用者角色對應
- **HTTP 方法**: `DELETE`
- **路徑**: `/api/v1/role-users/batch`
- **說明**: 批次刪除多筆使用者角色對應
- **請求格式**:
  ```json
  {
    "roleId": "string",
    "userIds": ["U001", "U002", "U003"]
  }
  ```

#### 3.1.4 匯出角色之使用者報表
- **HTTP 方法**: `POST`
- **路徑**: `/api/v1/role-users/export`
- **說明**: 匯出角色之使用者報表（Excel、PDF）
- **請求參數**:
  ```json
  {
    "roleId": "string",
    "exportFormat": "Excel|PDF"
  }
  ```
- **回應格式**: 檔案下載

```csharp
namespace RSL.IMS3.Application.Services

    public class RoleUserService : IRoleUserService
    {
        private readonly IRoleUserRepository _repository;
        private readonly ILogger<RoleUserService> _logger;

        public RoleUserService(
            IRoleUserRepository repository,
            ILogger<RoleUserService> logger)
        {
            _repository = repository;
            _logger = logger;
        }

        public async Task<RoleUserListResponseDto> GetRoleUserListAsync(
            RoleUserListRequestDto request,
            CancellationToken cancellationToken = default)
        {
            // 驗證角色是否存在
            var role = await _repository.GetRoleByIdAsync(request.RoleId, cancellationToken);
            if (role == null)
            {
                throw new NotFoundException($"角色 {request.RoleId} 不存在");
            }

            // 查詢使用者列表
            var queryResults = await _repository.GetRoleUsersAsync(
                request.RoleId,
                cancellationToken);

            // 組織回應資料
            var response = new RoleUserListResponseDto
            {
                RoleId = role.RoleId,
                RoleName = role.RoleName,
                Users = new List<RoleUserDto>()
            };

            foreach (var item in queryResults)
            {
                var user = new RoleUserDto
                {
                    UserId = item.USER_ID,
                    UserName = item.USER_NAME,
                    UserType = item.USER_TYPE,
                    UserTypeName = GetUserTypeName(item.USER_TYPE),
                    Status = item.STATUS,
                    StatusName = GetStatusName(item.STATUS),
                    Title = item.TITLE,
                    OrgId = item.ORG_ID,
                    OrgName = item.ORG_NAME
                };
                response.Users.Add(user);
            }

            return response;
        }
```

### 5.3 Repository: `RoleUserRepository.cs`
```csharp
namespace RSL.IMS3.Infrastructure.Data.Repositories
{
    public class RoleUserRepository : IRoleUserRepository
    {
        private readonly IDbConnection _dbConnection;

        public RoleUserRepository(IDbConnection dbConnection)
        {
            _dbConnection = dbConnection;
        }

        public async Task<Role?> GetRoleByIdAsync(string roleId, CancellationToken cancellationToken = default)
        {
            const string sql = @"
                SELECT RoleId, RoleName, RoleNote, Status, CreatedBy, CreatedAt, UpdatedBy, UpdatedAt
                FROM [dbo].[Roles]
                WHERE RoleId = @RoleId";

            var result = await _dbConnection.QueryFirstOrDefaultAsync<Role>(
                sql,
                new { RoleId = roleId },
                cancellationToken: cancellationToken);

            return result;
        }

        public async Task<IEnumerable<RoleUserQueryResult>> GetRoleUsersAsync(
            string roleId,
            CancellationToken cancellationToken = default)
        {
            const string sql = @"
                SELECT 
                    V.ROLE_ID,
                    V.ROLE_NAME,
                    V.USER_ID,
                    V.USER_NAME,
                    V.USER_TYPE,
                    V.STATUS,
                    V.TITLE,
                    V.ORG_ID,
                    O.OrgName AS ORG_NAME
                FROM [dbo].[V_RoleUsers] V
                LEFT JOIN [dbo].[Organizations] O ON V.ORG_ID = O.OrgId
                WHERE V.ROLE_ID = @RoleId
                ORDER BY V.USER_TYPE, V.USER_ID";

            var results = await _dbConnection.QueryAsync<RoleUserQueryResult>(
                sql,
                new { RoleId = roleId },
                cancellationToken: cancellationToken);

            return results;
        }

        public async Task DeleteRoleUserAsync(
            string roleId,
            string userId,
            CancellationToken cancellationToken = default)
        {
            const string sql = @"
                DELETE FROM [dbo].[UserRoles]
                WHERE RoleId = @RoleId AND UserId = @UserId";

            await _dbConnection.ExecuteAsync(
                sql,
                new { RoleId = roleId, UserId = userId },
                cancellationToken: cancellationToken);
        }

        public async Task BatchDeleteRoleUsersAsync(
            string roleId,
            List<string> userIds,
            CancellationToken cancellationToken = default)
        {
            const string sql = @"
                DELETE FROM [dbo].[UserRoles]
                WHERE RoleId = @RoleId AND UserId = @UserId";

            var parameters = userIds.Select(userId => new { RoleId = roleId, UserId = userId });
            await _dbConnection.ExecuteAsync(sql, parameters, cancellationToken: cancellationToken);
        }
    }
```
