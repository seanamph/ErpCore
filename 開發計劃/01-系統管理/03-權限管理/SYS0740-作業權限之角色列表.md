# SYS0740 - 作業權限之角色列表 開發計劃

## 一、功能概述

### 1.1 功能說明
- **功能代碼**: SYS0740
- **功能名稱**: 作業權限之角色列表
- **功能描述**: 查詢指定作業代碼下，哪些角色擁有該作業的權限，以及這些角色擁有的按鈕權限列表
- **參考舊程式**: 
  - `WEB/IMS_CORE/ASP/SYS0000/SYS0740_PR.ASP` (報表)
  - `WEB/IMS_CORE/ASP/SYS0000/SYS0740_FQ.asp` (查詢)
  - `IMS3/HANSHIN/RSL_CLASS/IMS3_SYS/MNG_USER_ROLE.cs` (GetProgRoleButton方法)

### 1.2 業務需求
- 支援依作業代碼查詢角色權限列表
- 顯示作業基本資訊（作業代碼、作業名稱）
- 顯示擁有該作業權限的角色列表（角色代碼、角色名稱）
- 顯示每個角色擁有的按鈕權限（按鈕名稱、頁面代碼）
- 支援報表列印功能
- 支援報表匯出功能（Excel、PDF）

```sql
CREATE TABLE [dbo].[Programs] (
```

```sql
CREATE TABLE [dbo].[Buttons] (
```

```sql
CREATE TABLE [dbo].[RoleButtons] (
    [TKey] BIGINT NOT NULL PRIMARY KEY IDENTITY(1,1),

-- 索引
CREATE NONCLUSTERED INDEX [IX_RoleButtons_RoleId] ON [dbo].[RoleButtons] ([RoleId]);
CREATE NONCLUSTERED INDEX [IX_RoleButtons_ButtonKey] ON [dbo].[RoleButtons] ([ButtonKey]);
```

### 2.2 查詢視圖 (View)

為了方便查詢，建議建立以下視圖：

#### 2.2.1 `V_ProgramRolePermission` - 作業角色權限視圖
```sql
CREATE VIEW [dbo].[V_ProgramRolePermission] AS
SELECT DISTINCT
    P.ProgramId AS PROG_ID,
    P.ProgramName AS PROG_NAME,
    R.RoleId AS ROLE_ID,
    R.RoleName AS ROLE_NAME,
    B.ButtonId AS BUTTON_ID,
    B.ButtonName AS BUTTON_NAME,
    B.PageId AS PAGE_ID
FROM [dbo].[Programs] P
INNER JOIN [dbo].[Buttons] B ON P.ProgramId = B.ProgramId
INNER JOIN [dbo].[RoleButtons] RB ON B.TKey = RB.ButtonKey
INNER JOIN [dbo].[Roles] R ON RB.RoleId = R.RoleId
WHERE RB.RoleId IS NOT NULL
  AND RB.UserId IS NULL
  AND B.Status = '1' 
  AND P.Status = '1';
```

### 5.2 Service: `ProgramRolePermissionService.cs`
```csharp
namespace RSL.IMS3.Application.Services
{
    public class ProgramRolePermissionService : IProgramRolePermissionService
    {
        private readonly IProgramRolePermissionRepository _repository;
        private readonly ILogger<ProgramRolePermissionService> _logger;

        public ProgramRolePermissionService(
            IProgramRolePermissionRepository repository,
            ILogger<ProgramRolePermissionService> logger)
        {
            _repository = repository;
            _logger = logger;
        }

        public async Task<ProgramRolePermissionListResponseDto> GetProgramRolePermissionListAsync(
            ProgramRolePermissionListRequestDto request,
            CancellationToken cancellationToken = default)
        {
            // 驗證作業是否存在
            var program = await _repository.GetProgramByIdAsync(request.ProgramId, cancellationToken);
            if (program == null)
            {
                throw new NotFoundException($"作業 {request.ProgramId} 不存在");
            }

            // 查詢權限資料
            var queryResults = await _repository.GetProgramRolePermissionsAsync(
                request.ProgramId,
                cancellationToken);

            // 組織角色與按鈕權限的對應關係
            var response = new ProgramRolePermissionListResponseDto
            {
                ProgramId = program.ProgramId,
                ProgramName = program.ProgramName,
                Roles = new List<RolePermissionDto>()
            };

            var roleDict = new Dictionary<string, RolePermissionDto>();

            foreach (var item in queryResults)
            {
                // 角色層級
                if (!roleDict.ContainsKey(item.ROLE_ID))
                {
                    roleDict[item.ROLE_ID] = new RolePermissionDto
                    {
                        RoleId = item.ROLE_ID,
                        RoleName = item.ROLE_NAME,
                        Buttons = new List<ButtonPermissionDto>()
                    };
                }

                // 按鈕層級
                var button = new ButtonPermissionDto
                {
                    ButtonId = item.BUTTON_ID,
                    ButtonName = item.BUTTON_NAME,
                    PageId = item.PAGE_ID
                };
                roleDict[item.ROLE_ID].Buttons.Add(button);
            }

            response.Roles = roleDict.Values.OrderBy(x => x.RoleId).ToList();
            return response;
        }

        public async Task<byte[]> ExportProgramRolePermissionReportAsync(

            string exportFormat,

            var data = await GetProgramRolePermissionListAsync(request, cancellationToken);

            if (exportFormat == "Excel")
            {
                return ExportToExcel(data);
            }
            else
            {
                return ExportToPdf(data);
```

### 5.3 Repository: `ProgramRolePermissionRepository.cs`
```csharp
namespace RSL.IMS3.Infrastructure.Data.Repositories
{
    public class ProgramRolePermissionRepository : IProgramRolePermissionRepository
    {
        private readonly IDbConnection _dbConnection;

        public ProgramRolePermissionRepository(IDbConnection dbConnection)
        {
            _dbConnection = dbConnection;
        }

        public async Task<Program?> GetProgramByIdAsync(string programId, CancellationToken cancellationToken = default)
        {
            const string sql = @"
                SELECT ProgramId, ProgramName, MenuId, ProgramNote, ProgramUrl, Status, SortOrder, CreatedBy, CreatedAt, UpdatedBy, UpdatedAt
                FROM [dbo].[Programs]
                WHERE ProgramId = @ProgramId AND Status = '1'";

            var result = await _dbConnection.QueryFirstOrDefaultAsync<Program>(
                sql,
                new { ProgramId = programId },
                cancellationToken: cancellationToken);

            return result;
        }

        public async Task<IEnumerable<ProgramRolePermissionQueryResult>> GetProgramRolePermissionsAsync(
            string programId,
            CancellationToken cancellationToken = default)
        {
            const string sql = @"
                SELECT 
                    V.PROG_ID,
                    V.PROG_NAME,
                    V.ROLE_ID,
                    V.ROLE_NAME,
                    V.BUTTON_ID,
                    V.BUTTON_NAME,
                    V.PAGE_ID
                FROM [dbo].[V_ProgramRolePermission] V
                WHERE V.PROG_ID = @ProgramId
                ORDER BY V.ROLE_ID, V.PAGE_ID, V.BUTTON_ID";

            var results = await _dbConnection.QueryAsync<ProgramRolePermissionQueryResult>(
                sql,
                new { ProgramId = programId },
                cancellationToken: cancellationToken);

            return results;
        }
```
