# SYS8000.ESKYLAND - 客戶定制模組系列 開發計劃

## 一、功能概述

### 1.1 功能說明
- **功能代碼**: SYS8000.ESKYLAND 系列
- **功能名稱**: ESKYLAND客戶定制租賃管理模組系列
- **功能描述**: 提供ESKYLAND客戶定制租賃管理功能，此模組為SYS8000的ESKYLAND客戶定制版本，功能類似但針對ESKYLAND客戶場景優化
- **參考舊程式**: 
  - `WEB/IMS_CORE/ASP/SYS8000.ESKYLAND/*.ASP` (ESKYLAND客戶定制租賃管理功能)

### 1.2 業務需求
- 管理ESKYLAND客戶特定的租賃資料
- 支援ESKYLAND客戶特定的租賃處理
- 符合ESKYLAND客戶業務流程
- 支援ESKYLAND客戶特定報表

---

## 二、資料庫設計 (Schema)

### 2.1 主要資料表

使用與SYS8000相同的資料表結構，但增加ESKYLAND特定的欄位和資料表。

#### 2.1.1 `Sys8000EskylandLeases` - ESKYLAND租賃主檔
```sql
CREATE TABLE [dbo].[Sys8000EskylandLeases] (
    [TKey] BIGINT NOT NULL PRIMARY KEY IDENTITY(1,1),
    [LeaseId] NVARCHAR(50) NOT NULL, -- 租賃編號
    [LeaseName] NVARCHAR(100) NOT NULL, -- 租賃名稱
    [EskylandSpecificField] NVARCHAR(200) NULL, -- ESKYLAND特定欄位
    [Status] NVARCHAR(10) NOT NULL DEFAULT 'A', -- 狀態 (A:啟用, I:停用)
    [CreatedBy] NVARCHAR(50) NULL,
    [CreatedAt] DATETIME2 NOT NULL DEFAULT GETDATE(),
    [UpdatedBy] NVARCHAR(50) NULL,
    [UpdatedAt] DATETIME2 NOT NULL DEFAULT GETDATE(),
    CONSTRAINT [UQ_Sys8000EskylandLeases_LeaseId] UNIQUE ([LeaseId])
);
```

## 三、後端 API 設計

### 3.1 API 端點列表

#### 3.1.1 查詢ESKYLAND租賃列表
- **HTTP 方法**: `GET`
- **路徑**: `/api/v1/sys8000-eskyland/leases`
- **說明**: 查詢ESKYLAND租賃列表

#### 3.1.2 新增ESKYLAND租賃
- **HTTP 方法**: `POST`
- **路徑**: `/api/v1/sys8000-eskyland/leases`
- **說明**: 新增ESKYLAND租賃資料

#### 3.1.3 修改ESKYLAND租賃
- **HTTP 方法**: `PUT`
- **路徑**: `/api/v1/sys8000-eskyland/leases/{leaseId}`
- **說明**: 修改ESKYLAND租賃資料

#### 3.1.4 刪除ESKYLAND租賃
- **HTTP 方法**: `DELETE`
- **路徑**: `/api/v1/sys8000-eskyland/leases/{leaseId}`
- **說明**: 刪除ESKYLAND租賃資料
