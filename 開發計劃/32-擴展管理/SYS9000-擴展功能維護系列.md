# SYS9000 - 擴展功能維護系列 開發計劃

## 一、功能概述

### 1.1 功能說明
- **功能代碼**: SYS9000 系列
- **功能名稱**: 擴展功能維護系列
- **功能描述**: 提供擴展功能資料的新增、修改、刪除、查詢功能，包含擴展基礎功能（SYS9105-SYS9180）、擴展處理功能（SYS9210-SYS9250）、擴展報表功能（SYS9310-SYS9670）、擴展查詢功能（SYS9510-SYS9540）等資訊管理
- **參考舊程式**: 
  - `WEB/IMS_CORE/ASP/SYS9000/SYS9105_FI.ASP` (新增)
  - `WEB/IMS_CORE/ASP/SYS9000/SYS9105_FU.ASP` (修改)
  - `WEB/IMS_CORE/ASP/SYS9000/SYS9105_FD.ASP` (刪除)
  - `WEB/IMS_CORE/ASP/SYS9000/SYS9105_FQ.ASP` (查詢)
  - `WEB/IMS_CORE/ASP/SYS9000/SYS9105_FB.ASP` (瀏覽)
  - `WEB/IMS_CORE/ASP/SYS9000/SYS9105_PR.ASP` (報表)

### 1.2 業務需求
- 管理擴展功能基本資料
- 支援擴展功能的新增、修改、刪除、查詢
- 支援批量新增功能
- 支援擴展功能的啟用/停用
- 支援擴展功能的排序
- 支援擴展功能的報表列印
- 支援擴展功能的背景處理
- 支援多版本管理

## 二、資料庫設計 (Schema)

### 2.1 主要資料表: `ExtensionFunctions` (對應舊系統 `SYS9_EXTENSION`)

```sql
CREATE TABLE [dbo].[ExtensionFunctions] (
    [TKey] BIGINT NOT NULL PRIMARY KEY IDENTITY(1,1), -- 主鍵 (T_KEY)
    [ExtensionId] NVARCHAR(50) NOT NULL, -- 擴展功能代碼 (EXTENSION_ID)
    [ExtensionName] NVARCHAR(100) NOT NULL, -- 擴展功能名稱 (EXTENSION_NAME)
    [ExtensionType] NVARCHAR(20) NULL, -- 擴展類型 (EXTENSION_TYPE, BASE:基礎, PROCESS:處理, REPORT:報表, QUERY:查詢)
    [ExtensionValue] NVARCHAR(500) NULL, -- 擴展值 (EXTENSION_VALUE)
    [ExtensionConfig] NVARCHAR(MAX) NULL, -- 擴展設定 (JSON格式) (EXTENSION_CONFIG)
    [SeqNo] INT NULL DEFAULT 0, -- 排序序號 (SEQ_NO)
    [Status] NVARCHAR(10) NOT NULL DEFAULT '1', -- 狀態 (STATUS) 1:啟用, 0:停用
    [Version] NVARCHAR(20) NULL, -- 版本號 (VERSION)
    [Notes] NVARCHAR(500) NULL, -- 備註 (NOTES)
    [CreatedBy] NVARCHAR(50) NULL, -- 建立者 (CREATED_BY)
    [CreatedAt] DATETIME2 NOT NULL DEFAULT GETDATE(), -- 建立時間 (CREATED_AT)
    [UpdatedBy] NVARCHAR(50) NULL, -- 更新者 (UPDATED_BY)
    [UpdatedAt] DATETIME2 NOT NULL DEFAULT GETDATE(), -- 更新時間 (UPDATED_AT)
    [CreatedPriority] INT NULL, -- 建立者等級 (CREATED_PRIORITY)
    [CreatedGroup] NVARCHAR(50) NULL, -- 建立者群組 (CREATED_GROUP)
    CONSTRAINT [PK_ExtensionFunctions] PRIMARY KEY CLUSTERED ([TKey] ASC),
    CONSTRAINT [UQ_ExtensionFunctions_ExtensionId] UNIQUE ([ExtensionId])
);

-- 索引
CREATE NONCLUSTERED INDEX [IX_ExtensionFunctions_ExtensionId] ON [dbo].[ExtensionFunctions] ([ExtensionId]);
CREATE NONCLUSTERED INDEX [IX_ExtensionFunctions_ExtensionType] ON [dbo].[ExtensionFunctions] ([ExtensionType]);
CREATE NONCLUSTERED INDEX [IX_ExtensionFunctions_Status] ON [dbo].[ExtensionFunctions] ([Status]);
CREATE NONCLUSTERED INDEX [IX_ExtensionFunctions_SeqNo] ON [dbo].[ExtensionFunctions] ([SeqNo]);
```

### 3.1 API 端點列表

#### 3.1.1 查詢擴展功能列表
- **HTTP 方法**: `GET`
- **路徑**: `/api/v1/extension-functions`
- **說明**: 查詢擴展功能列表，支援分頁、排序、篩選
- **請求參數**:
  ```json
  {
    "pageIndex": 1,
    "pageSize": 20,
    "sortField": "SeqNo",
    "sortOrder": "ASC",
    "filters": {
      "extensionId": "",
      "extensionName": "",
      "extensionType": "",
      "status": ""
    }
  }
  ```
- **回應格式**:
  ```json
  {
    "success": true,
    "code": 200,
    "message": "查詢成功",
    "data": {
      "items": [
        {
          "tKey": 1,
          "extensionId": "SYS9105",
          "extensionName": "擴展基礎功能1",
          "extensionType": "BASE",
          "extensionValue": "VALUE1",
          "extensionConfig": "{}",
          "seqNo": 1,
          "status": "1",
          "version": "1.0",
          "notes": "擴展功能說明"

      ],
      "totalCount": 100,
      "pageIndex": 1,
      "pageSize": 20,
      "totalPages": 5
    },
    "timestamp": "2024-01-01T00:00:00Z"
  }
  ```

#### 3.1.2 查詢單筆擴展功能
- **HTTP 方法**: `GET`
- **路徑**: `/api/v1/extension-functions/{tKey}`
- **說明**: 根據主鍵查詢單筆擴展功能資料
- **路徑參數**:
  - `tKey`: 主鍵
- **回應格式**: 同查詢列表的單筆資料格式

#### 3.1.3 根據擴展代碼查詢
- **HTTP 方法**: `GET`
- **路徑**: `/api/v1/extension-functions/by-id/{extensionId}`
- **說明**: 根據擴展功能代碼查詢擴展功能資料
- **路徑參數**:
  - `extensionId`: 擴展功能代碼
- **回應格式**: 同查詢列表的單筆資料格式

#### 3.1.4 新增擴展功能
- **HTTP 方法**: `POST`
- **路徑**: `/api/v1/extension-functions`
- **說明**: 新增擴展功能資料
- **請求格式**:
  ```json
  {
    "extensionId": "SYS9105",
    "extensionName": "擴展基礎功能1",
    "extensionType": "BASE",
    "extensionValue": "VALUE1",
    "extensionConfig": "{}",
    "seqNo": 1,
    "status": "1",
    "version": "1.0",
    "notes": "擴展功能說明"
  }
  ```
- **回應格式**:
  ```json
  {
    "success": true,
    "code": 200,
    "message": "新增成功",
    "data": {
      "tKey": 1
    },
    "timestamp": "2024-01-01T00:00:00Z"
  }
  ```

#### 3.1.5 修改擴展功能
- **HTTP 方法**: `PUT`
- **路徑**: `/api/v1/extension-functions/{tKey}`
- **說明**: 修改擴展功能資料
- **路徑參數**:
  - `tKey`: 主鍵
- **請求格式**: 同新增，但 `extensionId` 不可修改
- **回應格式**: 同新增

#### 3.1.6 刪除擴展功能
- **HTTP 方法**: `DELETE`
- **路徑**: `/api/v1/extension-functions/{tKey}`
- **說明**: 刪除擴展功能資料（軟刪除或硬刪除）
- **路徑參數**:
  - `tKey`: 主鍵
- **回應格式**:
  ```json
  {
    "success": true,
    "code": 200,
    "message": "刪除成功",
    "data": null,
    "timestamp": "2024-01-01T00:00:00Z"
  }
  ```

#### 3.1.7 批次新增擴展功能
- **HTTP 方法**: `POST`
- **路徑**: `/api/v1/extension-functions/batch`
- **說明**: 批次新增多筆擴展功能
- **請求格式**:
  ```json
  {
    "items": [
      {
        "extensionId": "SYS9105",
        "extensionName": "擴展基礎功能1",
        "extensionType": "BASE",
        "extensionValue": "VALUE1",
        "seqNo": 1,
        "status": "1"

      {
        "extensionId": "SYS9110",
        "extensionName": "擴展基礎功能2",
        "extensionType": "BASE",
        "extensionValue": "VALUE2",
        "seqNo": 2,
        "status": "1"
      }
    ]
  }
  ```

#### 3.1.8 啟用/停用擴展功能
- **HTTP 方法**: `PUT`
- **路徑**: `/api/v1/extension-functions/{tKey}/status`
- **說明**: 啟用或停用擴展功能
- **請求格式**:
  ```json
  {
    "status": "1" // 1:啟用, 0:停用
  }
  ```

### 4.3 API 呼叫 (`extensionFunction.api.ts`)
```typescript
import request from '@/utils/request';

export interface ExtensionFunctionDto {
  tKey: number;
  extensionId: string;
  extensionName: string;
  extensionType?: string;
  extensionValue?: string;
  extensionConfig?: string;
  seqNo?: number;
  status: string;
  version?: string;
  notes?: string;
}

export interface ExtensionFunctionQueryDto {
  pageIndex: number;
  pageSize: number;
  sortField?: string;
  sortOrder?: 'ASC' | 'DESC';
  filters?: {
    extensionId?: string;
    extensionName?: string;
    extensionType?: string;
    status?: string;
  };
}

export interface CreateExtensionFunctionDto {
  extensionId: string;
  extensionName: string;
  extensionType?: string;
  extensionValue?: string;
  extensionConfig?: string;
  seqNo?: number;
  status: string;
  version?: string;
  notes?: string;
}

export interface UpdateExtensionFunctionDto extends Omit<CreateExtensionFunctionDto, 'extensionId'> {}

export interface BatchCreateExtensionFunctionDto {
  items: CreateExtensionFunctionDto[];
}

// API 函數
export const getExtensionFunctionList = (query: ExtensionFunctionQueryDto) => {
  return request.get<ApiResponse<PagedResult<ExtensionFunctionDto>>>('/api/v1/extension-functions', { params: query });
};

export const getExtensionFunctionById = (tKey: number) => {
  return request.get<ApiResponse<ExtensionFunctionDto>>(`/api/v1/extension-functions/${tKey}`);
};

export const getExtensionFunctionByExtensionId = (extensionId: string) => {
  return request.get<ApiResponse<ExtensionFunctionDto>>(`/api/v1/extension-functions/by-id/${extensionId}`);
};

export const createExtensionFunction = (data: CreateExtensionFunctionDto) => {
  return request.post<ApiResponse<number>>('/api/v1/extension-functions', data);
};

export const updateExtensionFunction = (tKey: number, data: UpdateExtensionFunctionDto) => {
  return request.put<ApiResponse>(`/api/v1/extension-functions/${tKey}`, data);
};

export const deleteExtensionFunction = (tKey: number) => {
  return request.delete<ApiResponse>(`/api/v1/extension-functions/${tKey}`);
};

export const batchCreateExtensionFunctions = (data: BatchCreateExtensionFunctionDto) => {
  return request.post<ApiResponse>('/api/v1/extension-functions/batch', data);
};

export const updateStatus = (tKey: number, status: string) => {
  return request.put<ApiResponse>(`/api/v1/extension-functions/${tKey}/status`, { status });
};
```

### 6.1 安全性
- 必須實作權限檢查
- 敏感資料必須加密傳輸 (HTTPS)
- 批次操作必須限制數量

### 6.2 效能
- 大量資料查詢必須使用分頁
- 必須建立適當的索引
- 必須使用快取機制

### 6.3 資料驗證
- 擴展功能代碼必須唯一
- 必填欄位必須驗證
- JSON格式必須驗證

### 6.4 業務邏輯
- 刪除擴展功能前必須檢查是否有相關資料
- 停用擴展功能時必須檢查是否有進行中的業務
- 批次新增必須驗證資料完整性
